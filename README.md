# 🚗 Car Price Prediction (ML Project)

## 📌 Overview

This project is part of my machine learning learning journey where I implemented a complete regression pipeline to predict car prices.

## 🎯 Objective

To understand and apply core machine learning concepts such as:

* Data preprocessing
* Feature engineering
* Model training
* Model evaluation

## ⚙️ Workflow

1. Data Cleaning and Preprocessing
2. Feature Engineering (Car Age, Encoding)
3. Model Training (Linear Regression, Random Forest, XGBoost)
4. Performance Evaluation using R² Score

## 📊 Results

Achieved up to ~85–90% R² score using ensemble models.

## 🧠 What I Learned

* Importance of feature engineering
* Difference between linear and non-linear models
* Handling categorical variables
* Avoiding overfitting

## 🚀 Future Improvements

* Hyperparameter tuning
* Deploy model as a web app
* Use cross-validation

## 📂 Dataset

Ford Car Price Dataset
